# Aakkosmuistipeli
Ohjelmoinnin harjoitustyö

Aakkosmuistipelillä voit harjaannuttaa muistiasi, mikäli se reistailee!

Harjoitustyön tarkoituksena on siis ohjelmoida muistipeli, jossa etsittävät parit ovat kirjaimia.
